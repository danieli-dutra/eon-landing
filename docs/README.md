# 🎬 Director's Cut

Você já viu o resultado final.

Agora quero te convidar para conhecer as escolhas que tornaram essa experiência possível.

Este não é um making of.

Também não é uma documentação técnica.

É uma conversa sobre processo, intenção e decisões.

Porque, no fim, a EON nunca foi construída apenas para funcionar.

Ela foi construída para significar alguma coisa.

---

# 🎯 A missão

Toda Landing Page nasce com um objetivo.

Capturar atenção.

Gerar cliques.

Converter visitantes.

Coletar leads.

A EON também começou assim.

Mas, em algum momento do desenvolvimento, a pergunta mudou.

> **E se uma Landing pudesse ser lembrada não pelo que vende, mas pela experiência que proporciona?**

Foi a partir dessa pergunta que a EON deixou de ser apenas um projeto Front-end e passou a se tornar um estudo sobre narrativa, direção de arte e experiências digitais.

# ✨ Por que a EON é diferente?

A maioria das Landing Pages explica.

**A EON prefere revelar.**

Ela não tenta vender um produto.

Não pretende ser uma agência.

Não quer parecer um estúdio.

Não promove um filme.

Nem um jogo.

E, de alguma forma...

**poderia ser qualquer um deles.**

Porque sua verdadeira proposta nunca foi vender alguma coisa.

Foi despertar curiosidade.

---

# ⚛️ Desafio

A EON é um projeto em **React**.

Não apenas porque foi construída com React.

Mas porque espera provocar uma reação.

Pensar.

Refletir.

Pertencer.

Aceitar o convite.

---

# 🎞️ Conversas

Nenhum projeto nasce no vazio.

A EON também não.

Ela conversa com o cinema que respeita o silêncio.

Com jogos que confiam na inteligência do jogador.

Com o design editorial que entende o valor do espaço em branco.

Com marcas que preferem identidade à tendência.

Essas referências nunca foram copiadas.

Elas serviram apenas como ponto de partida para construir uma linguagem própria.

**A24** • **FromSoftware** • **Assassin's Creed** • **Apple** • **Linear**

# 🧩 Por trás das decisões

Existe um motivo para a marca só aparecer completamente no final.

Existe um motivo para os quatro símbolos conversarem entre si.

Existe um motivo para cada escolha tipográfica.

Existe um motivo para cada breakpoint.

Existe um motivo para o ritmo da narrativa.

Existe um motivo para o espaço em branco.

Existe um motivo para cada detalhe.

A EON foi construída acreditando que boas experiências não acontecem por acaso.

Se existe uma decisão, existe uma intenção por trás dela.

# 👩🏻‍💻 Quem dirigiu esta experiência

Meu nome é **Danieli Dutra**.

Sou estudante de Análise e Desenvolvimento de Sistemas e formada em Artes Digitais.

A EON nasceu justamente do encontro entre esses dois mundos.

Mais do que desenvolver uma Landing Page, este projeto foi uma oportunidade para explorar como desenvolvimento, design, narrativa, UX e direção de arte podem trabalhar juntos para construir experiências digitais memoráveis.

A EON foi minha maneira de responder a uma pergunta simples:

> **Como o desenvolvimento pode ser encantador sem deixar de ser funcional?**

Se você chegou até aqui, espero que tenha encontrado sua própria resposta.
